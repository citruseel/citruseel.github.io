The Recommender

An application to assist socially awkward people with conversing

In this project, my friends and I created a web app for HackWPI2022 that utilized AssemblyAI's voice-to-text API along with OpenAI's GPT-3 language model to provide short responses to given sentences. The intended target audience was similarly socially awkward engineering students, but the application could be expanded to anything which requires returning an automated response to a given audio input. In our application, audio was recorded as a json and sent to AssemblyAI's server which asynchronously converted the audio to text. The text was then processed with the Davinci model. Only the first 3 responses were selected as the output. This project won an award for
"Best Use of AssemblyAI"!
