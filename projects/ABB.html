ABB IRB140 Palletization

Palletize metal blocks with a 6 DoF industrial robot arm

In this project, we programmed the ABB IRB140 to stack 3 metal blocks in a hut-like configuration (such as shown below). Most of our work was done offline on ABB's RobotStudio programming software. Offline programming allowed us to see the targets we set graphically and create work objects to base the physical targets on without needing to be in the lab. Once the program was completed, the team fine-tuned the path planning and calibrated the coordinates of the work objects in the lab to run the demo.
